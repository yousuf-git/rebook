# ReBook

ReBook is an interactive landing page for an old books sale, purchase, and exchange platform. It features a vintage "Dark Academia" aesthetic and smooth scroll animations.

## Features

- Hero Section with animated elements
- How It Works section explaining the process
- Featured Collection grid with hover effects
- Client Reviews / Testimonials
- Newsletter subscription form
- Responsive design for all devices

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Setup and Usage

1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser.
3. No build step or server is required as it uses the Tailwind CSS Play CDN.
